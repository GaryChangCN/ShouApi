## 海大新媒体

上海海洋大学新型URP查询暨校园生活服务端（api端）

**目前前端只有小程序版** 
[海大新媒体小程序]("https://github.com/GaryChangCN/ShouMedia-wxapp")

[API文档](./api.md);

## 使用

```
git clone
npm install

npm run dev // dev
node app.js // prod
```

如果你不需要使用小程序接口，可以再app.js里面注释掉小程序接口部分，
如果使用则需要修改小程序配置文件在根目录的wxappConfig.js里，格式在wxappConfig.example.js中
